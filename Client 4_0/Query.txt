[] - обязательно
{} - опцианально

Общие ответы сервера:

Неизвестная ошибка на сервере
Not Found

1) Регистрация
api/registration/[login]&[password]

Проверяется уникальность логина и длина пароля( > 3 )

Ответ сервера
Успешная регистрация
registration/0

Такой пользователь уже существует
registration/1

Слишком короткий пароль
registration/2

Неизвестная ошибка на сервере
registration/3

2) Аутентификация => jwt token
api/authentication/[login]&[password]

Проверяется логин и пароль и возвращается токен

Ответ сервера
Успешная аутентификация
authentication/0/[{[login]&[jwt]}]

Неизвестная ошибка на сервере
authentication/3

Логин или пароль неверный
authentication/4

3) Получить список пользователей
api/getuserlist/[count]

count - ограничение по кол-ву пользователь(сервер может прислать меньше)
Возвращается список логинов
{login1, login2, .., loginN}

4) Запрос непрочитанных сообщений
api/unread/[login]&[jwt]

Проверяется логин и токен
Возвращаются все пользователи, от которых есть непрочитанные сообщения
{login1, login2, .., loginN}

5) Запрос сообщений от определённого пользователя
api/read/[login_root]&[jwt]/[login_user]/{count}

login_root - логин того, кто запрашивает
login_user - логин, чьи сообщения запрашивают
count - ограничение по кол-ву сообщений(сервер может прислать меньше)

{{"text1", data, status, 1/0}, {"text2", data, status, 1/0}, .., {"textN", data, status, 1/0},}

textI - Текст сообщения
data - дата отправки сообщения
status - Прочитано/Не прочитано
1 - Сообщение принадлежит login_root, 0 - login_user

6) Отправка сообщения пользователю
api/send/[login_root]&[jwt]/[login_user]/["text"]

login_root - логин того, кто отправляет
login_user - логин, кому отправляют

7) Групповая рассылка
api/groupsend/[login_root]&[jwt]/[count]/[login_user1]/[login_user2]/.../[login_userN]/["text"]

count == N - Кол-во пользователей
Сообщение отправляется в личные сообщения всем указанным пользователям

8) Отключение
api/exit/[login_root]&[jwt]

Удаление токена из базы
Отключение пользователя от сервера
